## Getting Started

### Prerequisites

- Node.js 14+ 
- npm or yarn
- Solana wallet (Phantom recommended)

### Installation

```bash
npm install
# or
yarn install
```

### Development

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the application.
